# SimpleAR_Unity

### Requirments
* Install Unity Hub " https://unity.com/pages/unity-pro-buy-now?gclid=Cj0KCQiAuvOPBhDXARIsAKzLQ8Fvxpt8Ng32RbNFCrb7G18xq8_s6Y9zy65O_KXsU5fDQ1dSu1iFZm8aAqY-EALw_wcB&gclsrc=aw.ds "
